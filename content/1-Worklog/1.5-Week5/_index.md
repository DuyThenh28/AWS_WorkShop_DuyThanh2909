## Weekly Objectives

This week focused on learning AWS networking by building a Virtual Private Cloud (VPC), configuring networking components, and deploying EC2 instances within a secure cloud network.

## Activities

| Day | Activity | Date |
|------|----------|------|
| Monday | Learned Amazon VPC architecture and AWS networking concepts. | 18/05/2026 |
| Tuesday | Created a VPC, configured CIDR blocks, and built Public and Private Subnets. | 19/05/2026 |
| Wednesday | Configured an Internet Gateway and Route Tables. | 20/05/2026 |
| Thursday | Configured Security Groups and Network ACLs. | 21/05/2026 |
| Friday | Deployed an EC2 instance inside the VPC and connected via SSH using VS Code. | 22/05/2026 |

## Learning Outcomes

- Understood Amazon VPC architecture.
- Created Public and Private Subnets.
- Configured Internet Gateway and Route Tables.
- Managed network security using Security Groups and NACLs.
- Successfully deployed EC2 inside a VPC.
