## Introduction

This worklog records my learning journey, practical activities, and project development throughout my internship at **Amazon Web Services Vietnam Co., Ltd.**

Over a period of **12 weeks**, I explored Amazon Web Services (AWS), learned cloud computing concepts, and applied AWS services to build and deploy a real-world **Saxophone Store Website**. During the internship, I also gained hands-on experience in software development, cloud deployment, teamwork, source code management, and technical documentation.

Each weekly report summarizes the objectives, completed tasks, and knowledge gained during that stage of the internship.

---

## Internship Timeline

### Phase 1 – AWS Fundamentals

- **Week 1:** Explored AWS Management Console, Amazon EC2, and basic cloud computing concepts.
- **Week 2:** Practiced working with Amazon S3, IAM, and AWS CLI to manage cloud resources.
- **Week 3:** Deployed a sample website on Amazon EC2 and learned Amazon VPC, CloudWatch, and Git.
- **Week 4:** Studied AWS Identity and Access Management (IAM), user permissions, and Multi-Factor Authentication (MFA).

### Phase 2 – Project Planning and System Design

- **Week 5:** Designed the cloud network architecture using Amazon VPC, Subnets, Internet Gateway, and Security Groups.
- **Week 6:** Analyzed project requirements and designed the architecture and database for the Saxophone Store Website.
- **Week 7:** Set up the development environment, initialized the frontend and backend projects, and configured GitHub for team collaboration.
- **Week 8:** Developed the website interface, implemented product-related features, and configured AWS infrastructure.

### Phase 3 – Development, Deployment, and Completion

- **Week 9:** Implemented the shopping cart, product management, and deployed the application on Amazon EC2 for testing.
- **Week 10:** Deployed the website using AWS Amplify, integrated APIs, authentication, payment services, AI, and notifications.
- **Week 11:** Completed the remaining features, optimized system performance, and prepared technical documentation.
- **Week 12:** Performed final testing, completed the internship report, delivered the project, and summarized the internship experience.
